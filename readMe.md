## 欢迎来到nova

1.nova-common
* 通用工具包

2.nova-database
* 数据库组件(动态数据源)

3.nova-design
* 设计模式

4.nova-framework
* 框架核心组件

5.nova-mail
* 邮件

6.nova-mongodb
* mongodb

7.nova-mq-active

8.nova-mq-rabbit

9.nova-pay
* 支付组件

10.nova-redis
* redis组件

11.nova-tools
* demo,java8,hutool测试类,guava测试类,vavr测试类

12.nova-xxl
* xxl-job执行器,mysql2.3.1sql文件,oracle2.1.0

13.nova-limit
* 接口限流

14.nova-validation
* 接口校验
