## 10.序列化

### 10.1谨慎实现Serializable

1. 序列化和反序列化，对象转为字节流、字节流转为对象的过程
2. 通过序列化之后，可以将对象编码，通过网络传输到其它应用上，再反序列化，实现远程调用
3. 实现序列化维护代价大

- 默认导出私有字段，违背了封装
- 可能导致bug、安全问题、序列化绕过了构造器

4. 什么情况下需要实例化

- 想把内存中的对象状态保存到文件中
- 套接字(ip:port)在网络上传送对象
- 通过RMI传输对象
